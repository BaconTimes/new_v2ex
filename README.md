# new_v2ex

Nextjs 练手项目，仿[v2ex](https://v2ex.com)，请勿商用。如果觉得本项目对你有帮助，麻烦给个star⭐️，非常感谢。

Live Demo: [https://v2ex.now.sh](https://v2ex.now.sh)

## Feature

- SSR 有利于SEO

- 响应式布局，支持桌面和手机显示，（IE<=11未测试，可能存在部分布局问题）

- 支持白色和暗黑主题
